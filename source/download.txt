Prerequisites
-------------

Software Requirements
~~~~~~~~~~~~~~~~~~~~~

.. tabs::

   hidden: true

   tabs:
     - id: osx
       name: macOS
       content: |

         |compass-short| requires the 64-bit version of macOS 10.10 or
         later and MongoDB 3.0 or later.

     - id: windows
       name: Windows
       content: |

         |compass-short| requires the 64-bit version of Microsoft
         Windows 7 or later and MongoDB 3.0 or later.

         .. note::

            The Windows installation of |compass| requires
            `Microsoft .NET Framework version 4.5 or later <https://www.microsoft.com/en-us/download/details.aspx?id=30653>`__.
            The |compass-short| installer prompts you to install the 
            minimum required version of the .NET framework if it is 
            not already installed on your system.

     - id: ubuntu
       name: Ubuntu
       content: |

         |compass-short| requires the 64-bit version of Ubuntu 14.04 or
         later and MongoDB 3.0 or later.

     - id: rhel
       name: RHEL
       content: |

         |compass-short| requires the 64-bit version of RHEL 7+ or
         later and MongoDB 3.0 or later.

Download Compass
----------------

.. admonition:: |compass-short| bundled with MongoDB 3.6 and later
   :class: important

   |compass-short| is included as part of the MongoDB Server download.
   If you choose to install |compass-short| when you install
   :dl:`MongoDB Server <enterprise>`, you can follow the tutorial in
   the :manual:`MongoDB manual </reference/program/install_compass/>`
   instead of this tutorial.

To download |compass-short|, you can use your preferred web browser.

.. tabs::

   hidden: true

   tabs:
     - id: osx
       name: macOS
       content: |

         1. Open the {+download-page+}.

         #. For :guilabel:`Version`, select your preferred 
            :ref:`edition <compass-and-community-eds>` of
            |compass| |compass-stable|. 

         #. For :guilabel:`Platform`, select 
            :guilabel:`OS X 64-bit (10.10+)`.


         #. Click :guilabel:`Download` to start downloading your
            selected edition of Compass. The |compass| installer is a ``.dmg`` disk
            image.

     - id: windows
       name: Windows
       content: |

         1. Open the {+download-page+}.

         #. For :guilabel:`Version`, select your preferred 
            :ref:`edition <compass-and-community-eds>` of
            |compass| |compass-stable|. 

         #. For :guilabel:`Platform`, select your preferred installation
            method for the Windows platform:

            - Select :guilabel:`Windows 64-bit (7+)` to download
              |compass| as an ``.exe``

            - Select :guilabel:`Windows 64-bit (7+) (MSI) to download 
              |compass| as an MSI package.

            - Select :guilabel:`Windows 64-bit (7+) (Zip) to download
              |compass| as a ``zip`` archive.


         #. Click :guilabel:`Download` to start downloading your
            selected edition of |compass|. 

     - id: ubuntu
       name: Ubuntu
       content: |

         1. Open the {+download-page+}.

         #. For :guilabel:`Version`, select your preferred 
            :ref:`edition <compass-and-community-eds>` of
            |compass| |compass-stable|. 

         #. For :guilabel:`Platform`, select 
            :guilabel:`Ubuntu 64-bit (14.04+)`.


         #. Click :guilabel:`Download` to start downloading your
            selected edition of |compass|.  The |compass| installer is a 
            ``.deb`` package.

     - id: rhel
       name: RHEL
       content: |

         1. Open the {+download-page+}.

         #. For :guilabel:`Version`, select your preferred 
            :ref:`edition <compass-and-community-eds>` of
            |compass| |compass-stable|. 

         #. For :guilabel:`Platform`, select 
            :guilabel:`RedHat 64-bit (7+)`.


         #. Click :guilabel:`Download` to start downloading your
            selected edition of |compass|.  The
            |compass| installer is a ``.rpm`` package.

Once you have downloaded the latest version of |compass| for your
operating system, continue to :doc:`/install`.
